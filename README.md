| ------- | --------------- |

| `09:00` | It's 9 O'clock  |
| `11:00` | It's 11 O'clock |

## ЁЯУМ `wish`, `want`, `desire`,`fancy` And `action`

`Wish` : **ркИркЪрлНркЫрк╛**

`Action` : **ркХрк░рлНркдрк╡рлНркп**

| I Wish to Action |
| ---------------- |
| I wish to ""     |
| I want to ""     |
| I desire to ""   |
| I fancy to ""    |

#### ЁЯФ║ рк╡рк╛ркХрлНркп рк░ркЪркирк╛ (Sentences)

ркорк╛рк░рлЗ ркЬрк╡рлБркВ ркЫрлЗ.

